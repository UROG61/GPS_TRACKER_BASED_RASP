# GPS_TRACKER
Raspberry pi ve python tabanlı ve ek olarak html çalışmamızında olduğu görselleştirilmiş harita üzerinde anlık konum takibi projemizin detaylı kod içeriğine yüklemiş olduğumuz kütüphanemizden erişim sağlayabilirsiniz.

Sevgili Güray Gürkan hocamızın desteğiyle hazırlamış olduğumuz projemizi İstanbul Kültür üniversitesi Elektrik Elektronik mühendisliği bölümü öğrencisi olarak ben Hacı Selim Altıntaş ve Grup Arkadaşım Adem Güler ile birlikte "GRADUATİON PROJECT" Dersi Adı altında tamamlamış bulunmaktayız.
